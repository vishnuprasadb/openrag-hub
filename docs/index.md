## Why OpenRAG-Hub?

Most teams want to:
- Chat with internal documents
- Avoid vendor lock-in
- Control their data
- Integrate with existing tools like Slack

## Key Features

- Retrieval-Augmented Generation (RAG)
- Web-based chat interface (local-first)
- Slack integration (`/ask`, `/upload`)
- File and Google Drive ingestion
- Pluggable LLM and vector database providers
- Docker-ready for easy distribution
